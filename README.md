# 23andMeThis is a repo for managing machine readable artifacts for the [23andMe APIs](https://www.23andme.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [23andMe APIs](https://www.23andme.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.23andme.com/)- [Developer](https://www.23andme.com/)- [Twitter](https://twitter.com/23andMe)- [GitHub](https://github.com/23andMe)The [APIs.json](https://github.com/api-evangelist/23andme/blob/master/apis.json) index provides links to all the relevant links for the 23andMe APIs, including the OpenAPI and Postman if they are present.
